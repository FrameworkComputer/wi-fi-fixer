# wii-fi-fixer
Linux Wi-Fi script for detecting and fixing common Intel wifi issues


## Detects and fixes:

- If the wifi radio is off.
- Restarts NetworkManager (tested on Ubuntu/GNOME).
- Airplane mode.
- If modules are not loaded - detects and load them.
